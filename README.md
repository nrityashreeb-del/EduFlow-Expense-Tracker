EduFlow: Student Expense Tracker

A full-stack web application built with Django to help students manage their daily finances.

Key Features:

CRUD Operations: Users can Create, Read, Update, and Delete expense records.

Secure Backend: Uses Django's ORM and SQLite for robust data management.

Security: Integrated CSRF protection for secure form submissions.

Responsive UI: Built with Bootstrap to ensure a seamless experience on both mobile and desktop.

Tech Stack: Python, Django, SQLite, HTML5, CSS3, Bootstrap.
